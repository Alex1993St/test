В консоли прописываем: 
php artisan queue:work
после чего при заходе на главную страницу запустятся очереди и в логах появится информация 

Также можно запустить команду 
php artisan parse:json
и если не было запущено очереди то запустить их 
php artisan queue:work

Удалил .env с gitignore чтобы видели все настройки: 

N_COUNT=10 - это сколько данных обработает с tasks.json 
QUEUE_CONNECTION=redis - использую чтобы задачи выполнялись параллельно
