В консолі прописуємо:
php artisan queue:work
після чого при заході на головну сторінку запустяться черги і в логах з'явиться інформація

Також можна запустити команду
php artisan parse:json
і якщо не було запущено черги то запустити їх
php artisan queue:work

Видалив .env з гіта щоб бачили всі налаштування:

N_COUNT=10 - це скільки даних обробить з tasks.json
QUEUE_CONNECTION=redis - використовую щоб задачі виконувались паралельно
